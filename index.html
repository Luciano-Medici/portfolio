Etch-a-Sketch

Sobre este proyecto

Este proyecto es una aplicación web interactiva inspirada en el clásico Etch-a-Sketch, desarrollada como parte del curso Foundations de The Odin Project, utilizando HTML, CSS y JavaScript puro. El desafío principal consistía en construir una cuadrícula dinámica para dibujar al pasar el cursor, y una de las condiciones del ejercicio era implementar el layout únicamente con Flexbox, sin utilizar CSS Grid, con el objetivo de afianzar su uso y comprender mejor su comportamiento en el diseño de interfaces. Además de trabajar con manipulación del DOM y event handling, el proyecto incluye funciones como cambio de color, borrado del dibujo y ajuste del tamaño de la cuadrícula. Esto me permitió practicar lógica condicional, ciclos y una estructura de código clara y funcional.

Rock Scissors Paper Game

Sobre este proyecto

Este proyecto es una implementación del clásico juego Piedra, Papel o Tijera, desarrollado con HTML, CSS y JavaScript, como parte del curso Foundations de The Odin Project. En una primera etapa, el juego fue creado para funcionar desde la consola del navegador, lo que me permitió practicar conceptos fundamentales de lógica de programación, estructuras condicionales, funciones y bucles en JavaScript. Más adelante, integré una interfaz gráfica simple utilizando manipulación del DOM, lo que permitió al usuario interactuar con el juego a través de botones en lugar de comandos en consola. También incorporé un sistema de puntuación y una lógica de control para finalizar la partida al alcanzar cierto puntaje. Este proyecto fue clave para reforzar mis conocimientos de JavaScript puro, y para empezar a vincular la lógica con elementos visuales del navegador.

Landing page

Sobre este proyecto

Este proyecto es una landing page desarrollada como parte del curso Foundations de The Odin Project, utilizando únicamente HTML y CSS. El objetivo principal era aplicar conceptos fundamentales de maquetado web, priorizando el uso de Flexbox para organizar el diseño de forma responsiva y semántica. El ejercicio consistía en replicar una landing page siguiendo una maqueta de referencia, poniendo en práctica la estructura básica de una página moderna: header, sección principal, bloques de contenido, llamados a la acción y un footer. Me enfoqué en lograr una buena alineación, jerarquía visual y coherencia en los estilos. Este proyecto fue clave para afianzar mi manejo de selectores CSS, tipografía, espaciado y el uso de clases reutilizables para mantener un código limpio y ordenado.
